# Temporal Resource Allocation System

Projekt oparty na **React + TypeScript + Vite + SCSS**.

---

## Instalacja i uruchomienie

Aby uruchomić projekt lokalnie, upewnij się, że masz zainstalowane **Node.js (v18 lub nowszy)**.

### 1. Pobranie repozytorium

```bash
git clone https://github.com/JakubJurkian/temporal-resource-allocation-system.git
cd temporal-resource-allocation-system
```

### 2. Instalacja zależności

```bash
npm install
```

### 3. Uruchomienie serwera deweloperskiego

```bash
npm run dev
```

Aplikacja będzie dostępna pod adresem:  
**http://localhost:5173/** (lub adresem podanym w terminalu)

---

## Dostępne skrypty

| Komenda            | Działanie |
|--------------------|-----------|
| `npm run dev`       | Uruchamia tryb deweloperski |
| `npm run build`     | Buduje aplikację do `dist/` |
| `npm run preview`   | Podgląd wersji produkcyjnej |
| `npm run lint`      | Sprawdza kod za pomocą ESLint |
| `npm run storybook` | Uruchamia dokumentację komponentów |
| `npm test`          | Uruchamia testy jednostkowe (Vitest) |
